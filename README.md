# Projet Deep Learning - WikiArt

## Introduction

Ce projet consiste à développer un outil permettant d'obtenir automatiquement une description à partir d'une photo d'une oeuvre d'art.
Il s'agit de décrire le genre artistique, influence et éventuellement artiste.

Le modèle de Deep Learning est entraîné sur la base de données WikiArt disponible sur Kaggle.

## Installation

```bash
pip install -r requirements.txt
```
